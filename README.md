### <span id="Guia">⚡ Guia de início</span>

Antes de tudo, esteja certo de que você tenha o NodeJS e o Yarn instalado em sua máquina.

```bash
# Vá até a pasta
$ cd joaocouto-blog/

# Instale as dependências
$ yarn install

# Execute a aplicação
$ yarn dev
```
